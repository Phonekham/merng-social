# merng-social

Built on MERN stack with GraphQL.
## Features

### MVP

- Sign up / sign in 
- Post / Like Post / Comment On Post

## Topics

- Schema design

- Validation

- Authorization (access control)

- State management

- Real-time updates

## Technologies

### Back-End

- Node + Express

- MongoDB + Mongoose ORM

- Apollo Server v2

- graphql / graphql-tag

- jsonwebtoken

- bcryptjs

### Front-End

- React 16+

- Apollo Client v2

- Semantic-UI

- moment

- jsdecode
